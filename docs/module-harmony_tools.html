(static) processHarmonyHubs(modules, devices, type)

Processes Harmony Hubs during program setup, this function will connect to all Harmony hubs in the modules file and add the devices they control to both the modules object and the devices array
Parameters:
Name Type Description
modules Object modules object from the main script, will store harmony functions in it
devices Array.<Object> devices array from the main script, will add harmony devices to the array
type Object the Harmony object loaded from the modules.json file (this is obtained during the initial forEach loop in the processModules function in server.js)
